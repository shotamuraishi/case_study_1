# Case Study 1: Logistic Regression on COVID-19 Patient Data

## Overview
This project analyzes COVID-19 patient health data to build a logistic regression model that predicts patient outcomes.

The goal of this project is to deepen our understanding of logistic regression, including:
- Data preprocessing
- Feature selection
- Model training
- Model evaluation

## Dataset
The dataset is sourced from Kaggle:
COVID-19 Mexico Patient Health Dataset
https://www.kaggle.com/datasets/riteshahlawat/covid19-mexico-patient-health-dataset/data

## Methods
- Data cleaning and preprocessing
- Handling missing values
- Feature engineering
- Logistic Regression model implementation with regularization
- Model evaluation using:
  - Accuracy
  - Confusion Matrix
  - AIC
  - BIC

## Results
- Accuracy: 0.72 - 0.74
